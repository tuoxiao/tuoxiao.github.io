本地Github托管

2021-02-11 学习 Github 0 评论

本实例将展示如何将本地文件、程序上传到Github托管。

安装Git客户端

1、点击Git下载最新版Git客户端

2、直接默认安装，无需额外配置

3、将Git加载到右键

使用Git GUI 上传

1、如图所示生成新的仓库



2、按照图中所示步骤依次操作



3、点击stage changed,后会变成这样



4、填写Github仓库链接，注意transfer options中的选项必须选择一个，否则无法上传成功



5、上传成功后显示如图所示:



6、此时到GitHub仓库即可看到上传的文件
